API Reference

A simple guide to our available API endpoints.

GET /api/v1/ping

Check if the API service is running.

GET /api/v1/items

Retrieve a list of all available items.

GET /api/v1/items/{itemId}

Fetch a single item by its unique ID.

POST /api/v1/items

Create a new item with data in the request body.
