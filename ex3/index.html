API Documentation

Aapke API ke liye simple aur saaf documentation.

Available Endpoints

GET /api/users

Saare Users Hasil Karein

Yeh endpoint aapko database mein maujood sabhi users ki list dega.

GET /api/users/{id}

Ek User Hasil Karein

Kisi khaas user ki details hasil karne ke liye uski ID provide karein.

POST /api/users

Naya User Banayein

Request body mein user data bhej kar ek naya user create karein.

PUT /api/users/{id}

User Ko Update Karein

Kisi maujooda user ki details ko update karein.
